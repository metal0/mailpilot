# Welcome to Mailpilot

Mailpilot is an AI-powered email processing daemon that automatically classifies, organizes, and processes your emails using LLM intelligence.

## Features

- **LLM-Powered Processing**: Use OpenAI, Anthropic, Ollama, or any local model
- **Privacy-Focused**: Runs entirely on your infrastructure
- **Open Source**: Fully customizable and extensible
- **Multi-Provider Support**: Gmail, Outlook, Yahoo, iCloud, ProtonMail, and generic IMAP
- **Flexible Actions**: Move, flag, mark as read, or trigger custom workflows
- **Web Dashboard**: Monitor and manage your email processing
- **RESTful API**: Integrate with your existing tools
