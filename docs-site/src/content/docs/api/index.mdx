## Base URL

```
http://localhost:8080/api
```

## Rate Limits

Currently, there are no enforced rate limits on API keys. However, best practices recommend:

- Max 100 requests/minute for normal operations
- Max 10 requests/second for real-time monitoring

## Response Format

All API responses return JSON:

```json
{
  "success": true,
  "data": { ... }
}
```

Errors return:

```json
{
  "success": false,
  "error": "Error message"
}
```
